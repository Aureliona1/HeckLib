# Geometry properties
The following parameters can be used in geometry shapes.
- `Shape.Capsule`
- `Shape.Cube`
- `Shape.Cylinder`
- `Shape.Plane`
- `Shape.Quad`
- `Shape.Sphere`
- `Shape.Triangle`
  
# Shader properties
The following parameters can be used in material shaders
- `Shader.Standard`
- `Shader.OpaqueLight`
- `Shader.TransparentLight`
